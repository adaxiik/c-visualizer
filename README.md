# VisualizerBP

## Prerequisites
- Bash
- Node.js
- npm
- Parcel.js
- FabricJS

Optional:
- Docker

## Usage
- Set `mainScript.sh` to be executable with: `chmod +x ./buildOrRunScript.sh`
- Run `mainScript.sh` (and choose the desired option)

___
**Note**
(The site can be then found at http://localhost:1234)
